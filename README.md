# Hangman

This is lab 10 for CS202: Hangman CLI - A 2-player game

# How to run

g++ hangman.cpp -o hangman
./hangman

When prompted for a phrase, type out a phrase and press enter. Then (a different player) types a letter to guess that might be in that phrase. Then you can keep guessing that same way. Once you 'know' the phrase, you enter '0' and type in that EXACT phrase. 6 total incorrect letter guesses until lost, 1 incorrect phrase guess until lost, win if the guessing player guesses all the letters correct or correct exact phrase.
